AddEventListener

Con este método se puede agregar eventos a componentes html desde el código JS y sin la
necesidad de declararlos en la etiqueta HTML. Dicho método recibe dos parámetros.

  • El evento que se le desea agregar al componente.
  • Una función que se disparará al escuchar dicho evento.
