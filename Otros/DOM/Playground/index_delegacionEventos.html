Delegación de eventos ¿Que carajo es?


Se trata de una técnica mediante la cual es posible definir un evento para un elemento externo ,como ser un formulario,
y que este evento se propague a componentes internos. ¿Como se logra?. Facil, ahi van los pasos:

    Ejemplo con un formulario que tiene dos campos tipo texto.

  1. Asignar id al formulario.
  2. Desde el código JS recuperar el formulario. Puede ser utilizando el id de este si es que lo definiste
  3. Agregar un eventlistener, por ejemplo para el evento focus.
  4. Normalmente, agregarías 2 parámentros al addEventListener. Ahora, vas a agregar un tercero que será un campo booleando en true
  5. Definí que va a hacer la función, obvio, si no vas a hacer foco sobre el campo y no va a hacer un carajo.
  6. Finalmente, probalo. Ya esta. Al hacer foco sobre los campos nombre o apellido vas a llamar a la función que definiste para el evento focus.
